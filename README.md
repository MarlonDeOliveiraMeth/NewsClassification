# Classificação de Notícias

Projeto final da disciplina Deep Learning, parte da minha pós-graduação em Ciência de Dados e Machine Learning na PUC Campinas.

O estudo feito em Jupyter Notebook com a biblioteca Python scikit-learn, analisa títulos de notícias e classifica eles de acordo com suas categorias (esportes, tecnologia, saúde, etc).
